# 企业微信配置指南

本文档说明如何配置企业微信机器人以连接到 OpenCode Bridge。

---

## 1. 前置条件

- OpenCode Bridge 服务已部署
- OpenCode 已安装并运行
- 企业微信企业账号

---

## 2. 创建企业微信应用

### 步骤 1：登录管理后台

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/)
2. 进入"应用管理"

### 步骤 2：创建应用

1. 点击"应用" → "自建"
2. 创建新应用或选择已有应用
3. 记录 AgentId（Bot ID）和 Secret

---

## 3. Bridge 配置

### Web 面板配置

在 Web 配置面板（`http://localhost:4098`）中：

1. 进入"平台接入" → "企业微信（WeCom）"配置
2. 将"是否启用企业微信适配器"设置为 `true`
3. 填写企业微信 Bot ID（AgentId）
4. 填写企业微信 Secret
5. 保存配置

### 配置参数

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `WECOM_ENABLED` | 否 | `false` | 是否启用企业微信适配器 |
| `WECOM_BOT_ID` | 是 | - | 企业微信 Bot ID（AgentId） |
| `WECOM_SECRET` | 是 | - | 企业微信 Secret |
| `WECOM_SHOW_THINKING_CHAIN` | 否 | - | 显示 AI 思维链 |
| `WECOM_SHOW_TOOL_CHAIN` | 否 | - | 显示工具调用链 |

---

## 4. 配置消息接收

### 设置 API 接收地址

1. 在应用详情页面，找到"接收消息"配置
2. 设置 API 接收地址为：
   ```
   http://your-server:your-port/wecom/webhook
   ```
3. 保存配置

---

## 5. 配置权限

在企业微信管理后台，确保应用具有以下权限：

- 发送消息到用户/部门/标签
- 读取用户信息
- 管理通讯录（可选）

---

## 6. 使用方式

### 可用命令

| 命令 | 说明 |
|------|------|
| `/help` | 查看帮助 |
| `/panel` | 打开控制面板 |
| `/model <provider:model>` | 切换模型 |
| `/agent <name>` | 切换 Agent |
| `/session new` | 开启新话题 |
| `/undo` | 撤回上一轮交互 |
| `/compact` | 压缩上下文 |

---

## 7. 故障排查

### 机器人无响应

1. 检查 `WECOM_ENABLED` 是否为 `true`
2. 检查 `WECOM_BOT_ID` 和 `WECOM_SECRET` 是否正确
3. 检查消息接收地址是否正确配置
4. 查看服务日志中的错误信息

### 消息发送失败

1. 检查应用权限是否足够
2. 检查用户/群组 ID 是否正确
3. 检查网络连接是否正常

---

## 8. 注意事项

- 企业微信适配器目前支持文本消息交互
- 不支持富文本卡片（与飞书不同）
- 文件发送功能受限于企业微信 API
- 建议在测试群组中先验证配置
